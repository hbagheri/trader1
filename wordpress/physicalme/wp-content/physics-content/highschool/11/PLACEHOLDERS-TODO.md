# 📋 وضعیت — فیزیک یازدهم

## ✅ فصل ۱ — کامل
۱۲ مقاله + ۱۲ کوییز + ۶ شبیه‌ساز + آرایه‌ی ۳۲ مسئله (جدا).

## ✅ فصل ۲ — همه‌ی زیرفصل‌ها آماده
- [x] ۲-۱ جریان الکتریکی (current-flow)
- [x] ۲-۲ مقاومت و قانون اهم (ohm-law)
- [x] ۲-۳ عوامل مؤثر بر مقاومت / مقاومت ویژه (resistivity)
- [x] ۲-۴ نیروی محرکه و مدارها (emf-circuit)
- [x] ۲-۵ توان در مدارهای الکتریکی (power-meter)
- [x] ۲-۶ ترکیب مقاومت‌ها سری/موازی (combo-resistors)
- [ ] مسائلِ پایانِ فصل ۲ (ویجتِ حل مسائل + آرایه‌ی JS، مثلِ فصل ۱)

## 💡 اختیاری
- غنی‌سازیِ ۲-۳ با انواع مقاومت (رئوستا/پتانسیومتر)، کدِ رنگیِ کامل، ترمیستور، LDR.

## 🖼️ عکس‌ها
- ۲-۲: مقاومت رنگی، مولتی‌متر | ۲-۳: کابل پرفشار، ابررسانا، نیکروم
- ۲-۴: ماهیِ الکتریکی، باتری | ۲-۵: کنتورِ برق، LED vs رشته‌ای | ۲-۶: ریسه‌ی سری/موازی | ۱-۱: کهربا---
